# Tests

Esta carpeta agrupa las pruebas del proyecto a nivel monorepo.

Estructura prevista:

- `e2e/` para flujos Playwright;
- `fixtures/` para datos de prueba compartidos;
- `mocks/` para dobles de prueba cuando hagan falta.

Estado actual:

- la carpeta existe como base de organizacion;
- las pruebas E2E reales aun no estan implementadas;
- se corrigio la convencion `2e2` a `e2e` para seguir el nombre comun del ecosistema.
